名前: srr.sh srs.sh


概要: ストリーミングラジオを保存(srr.sh)、再生(srs.sh)するシェルスクリプト集


使用方法(srr.sh):

  ラジコ、らじる、AG、AFNを保存:  srr.sh -c チャンネル -t 分 -i -m メモ -o /出力/ディレクトリ/名前

    -c チャンネル  指定できるチャンネルは、
                   ラジコID(HOUSOU-DAIGAKU等)、NHK1、NHK2、NHKFM、AG、AFN

    -t 分  保存時間を分で指定

    -i(省略可能)  番組情報を取得

    -m メモ  ログに出力する任意のメモを指定
             * このオプションは省略可能

    -o /出力/ディレクトリ/名前  srsr.confで設定している$SRSR_OUTDIR以外に、保存ファイルを出力。
                                `/出力/ディレクトリ/'でディレクトリのみ指定、
                                `/出力/ディレクトリ/ファイル名'でディレクトリとファイル名を指定
                                * このオプションは省略可能

    -M(省略可能)  標準ファイル名(チャンネル名_年月日-時日)の`時'を、00時→24時、05時→29時にする

  保存テスト:  srr.sh -T チャンネル

    -T チャンネル  3分間保存。`srr.sh -c チャンネル -t 3 -i'を指定するのと同じ
                   チャンネルは省略可能
                   その場合、srsr.confのSRR_OMCHに設定したチャンルを保存

  コミュニティーFMを保存:  srr.sh -C URL -f コーデック -t 分

    -C URL  `cfmurl.sh -l'→`p チャンネル'で表示される、-C `URL'を指定

    -f コーデック  `cfmurl.sh -l'→`p チャンネル'で表示される、-f `コーデック'を指定
                   コーデックを省略した場合、`wma'

  ヘルプ:  -h


使用方法(srs.sh):

  ラジコ、らじる、AG、AFNを再生:  srs.sh -c チャンネル

  コミュニティーFMを再生:  srs.sh -C チャンネル

    再生中のオプション:
      r  インスタント保存
      s  インスタント保存停止
      b  番組名表示
      k  曲名表示(曲名配信しているラジコ局のみ)
      c  チャンネル変更
         チャンネルIDかヒストリー番号で指定
         `b'を指定すると、現在配信中のラジコとらじるの番組一覧
      q  終了
      h  再生中のヘルプを表示
